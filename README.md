# Credit Default ML

Proyecto de Machine Learning para predecir riesgo de default (incumplimiento de pago) usando el dataset de Home Credit (Kaggle).

## Objetivo
Construir un modelo de clasificación que estime la probabilidad de default y apoyar decisiones de crédito.

## Tecnologías
- Python
- Jupyter
- Scikit-Learn
- XGBoost / LightGBM
